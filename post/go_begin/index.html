开始学习Go--语法基础

May 27, 2022
1 min read
Go

语法基础

Go 是一种注重简单性和速度的编程语言。Go 语言的目标：

  • 快速编译
  • 不太笨重的代码（语法简单）
  • 自动释放未使用的内存（垃圾回收）
  • 易于编写同时执行多个操作的软件（并发\协程）
  • 很好的支持多核处理器

Go 程序代码结构

package main

import "fmt"

func main(){
    fmt.Println("Hello,Go!")
}

上面是典型的Go文件代码布局，主要分三部分：

  • package main 子句，表示该代码文件中的所有其余代码都属于 main 包。
  • import "fmt" 子句，表示我们导入了 fmt 包，可以使用里面公开的文本格式代码。
  • 实际代码 每个Go文件的最后一部分都是实际代码，它通常被分割成一个或多个 function(函数)。main 包和main函数是 Go 程序的入口点。

调用函数

调用函数的格式是：包名.函数名，比如 fmt.Println(),一个包中的函数可被其它包调用，前提是该函数是大写字母开头，表示是可导出的。

在代码中要使用其它包中的函数，首先要导入其它包，通过impport 子句。

import (
    "fmt"
    "math"
)

上面这种格式表示导入了两个包。

字符串

字符串是一系列字节，通常表示文本字符，通过字面量定义字符：双引号之间的文本，Go将把他们视为字符串,"Hello,Go"。

在字符串中，换行符、制表符和其它难以包含在程序中的字符可以用转义序列来表示：反斜杠(\)后跟表示另一个字符的字符。

转义序列 值
\n 换行符
\t 制表符
\" 双引号
\|反斜杠

符文

字符串通常用于表示一系列文本字符，而Go的符文(rune)则用于表示单个字符。 字符串字面量由双引号(")包围，单 rune 字面量由单引号(’)包围。 Go 使用 Unicode 标准来存储rune。rune被保存为数字代码，而不是字符本身。

类型

周围的事物通常都可以根据它们不同的用途分为不同的类型。同样，Go中的值都被划分为不同的类型，这些类型指定了这些值得用途。 Go 是静态类型的，在程序运行之前就知道了值的类型，也就是说在程序编译阶段就确定了所有值得类型。 可以通过reflect包的TypeOf函数来查看任何值的类型：

package main

import (
  "fmt"
  "reflect"
)
func main(){
  fmt.Println(reflect.TypeOf(100)) //int
  fmt.Println(reflect.TypeOf("Hello,Go!")) //string
}

声明变量

在 Go中，变量是包含值得一块存储，可以使用变量声明为变量命名。只需使用var关键字，后跟所需的名称以及将保存变量的值的类型。

var quantity int  //var 为声明变量的关键字，quantity 为变量名，int 为 保存变量的值得具体类型
var length,width float64 //可以一次声明同一类型的多个变量
var customerName string

一旦声明了一个变量，就可以用 =为它分配该类型的任何值。

quantity=2
customerName="alvinsu"

可以在同一语句中为多个变量赋值，只需将多个变量名放在=号的左边，将相同数量的值放在右边，并使用逗号分隔。

length,width=1.2,2.4

如果事先知道变量的值是什么，就可以在声明变量的时候同时赋值：

var quantity int = 4
var length,width float64 = 1.2,2.4
var customerName string = "alvinsu"

在声明变量的同时为其赋值，通常可以在声明中省略变量类型。

var quantity = 4
var length,width = 1.2,2.4
var customerName = "alvinsu"

零值

如果声明一个变量而没有给它赋值，该变量将包含其类型的零值。对于数值类型，零值就是 0，字符串变量的零值是空字符串，布尔类型的零值是 false。

短变量声明

在声明变量的时候就知道它的初始值是什么，那么可以使用短变量声明::=。一般短变量声明用在函数体内。

package main

import "fmt"

func main(){
  quantity := 4
  length,width := 1.2,2.4
  customerName := "alvinsu"

  fmt.Println(quantity)
  fmt.Println(length*width)
  fmt.Println(customerName)
}

命名规则

Go 有一套简单的规则，适用于变量、函数和类型的名称：

  • 名称必须以字母开头，并且可以有任意数量的额外的字母和数字。
  • 如果变量、函数和类型的名称以 大写 字母开头，则认为它是导出的，可以从当前包之外的包访问它。
  • 如果一个名称由多个单词组成，那么它的第一个单词之后的每个单词都应该首字母大写，并且它们应该连接在一起，中间没有空格： 
    customerName

转换

Go中的数学运算和比较运算要去包含的值具有相同的类型。如果参与计算的值是不同类型的，解决方案是使用 转换,它允许将类型转换为另一种类型。具体语法如下：

var myInt int = 2
var myFloat64 float64 = float64(myInt)
